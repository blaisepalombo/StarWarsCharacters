# StarWarsCharacters

This project is a web API that stores and manages a database of Star Wars characters. The application organizes characters into multiple collections and allows users to view, create, update, and delete character records. It is designed to help fans explore and organize characters from the Star Wars universe in a structured way. The project demonstrates RESTful API design, database integration, and API documentation using Swagger.
